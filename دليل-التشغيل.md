# MZ.S-ERP - Quick Start Guide

## How to Run

### Double-click: start.bat

Or from command prompt:
```bash
.\start.bat
```

## What It Does

1. Checks Node.js
2. Creates .env file (if needed)
3. Installs dependencies
4. Generates Prisma Client
5. Creates/checks database
6. Starts Backend + Frontend

## Access URLs

- Frontend: http://localhost:5173
- Backend: http://localhost:3000
- Health: http://localhost:3000/api/health

## Default Login

- Username: admin
- Password: admin123

## To Stop

Close both opened windows or press Ctrl+C in each.

## Troubleshooting

### System stops immediately?
```bash
rmdir /s /q node_modules
rmdir /s /q backend\node_modules
npm install
cd backend && npm install
cd ..
.\start.bat
```

### Port 3000 in use?
Close other apps or change PORT in .env

### White page?
Wait for startup then press Ctrl+F5
